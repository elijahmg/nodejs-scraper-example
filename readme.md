# Express server to get Meta and OEmbed data if presented

## Setup

```bash
# install packages
yarn

# Run application
yarn dev
```

## Test

Open browser and go to `http://localhost:8080?url=${URL_TO_PARSE}`

- Example ``http://localhost:8080?url=https://open.spotify.com/artist/6mdiAmATAx73kdxrNrnlao``
